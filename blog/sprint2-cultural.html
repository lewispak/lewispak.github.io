Add HTML and CSS to your Blog Challenge

Write a small story about HTML and CSS and their relationship. You could personify elements or speak from your own personal experience. The story should capture insight about the nature of HTML and CSS.

A really helpful video tutorial I came across was of one that described HTML and CSS as a builder and artist (for the sake of this post, I'm going to CSS the designer.) Basically, the way HTML and CSS were explained through this analogy is that HTML built the house while CSS designed it. In other words, HTML (the builder) built the structure and foundations of the site we were building, while CSS (the designer) decorated and furnished what was built. For example, let's say you wanted to build a brand new house. In order to do that, you'd have a builder (HTML) build the entire thing, while the decorators (CSS) would paint, style, and furnish the place.

Any moments of surprise or joy?

There was a moment of clarity about the course which reduced overwhelm and simplified the entire study process for me. That made me very joyful.

If HTML and CSS were animals, which animals do you think they would be? Why?

If HTML and CSS were animals, HTML would be a work-horse while CSS would be a Peacock. The reason for this is because work-horses are exactly that. They're raised and trained to do a specific set of task in order to fulfill jobs, increase productivity, and get work done. I believe this relates to HTML because without it, work wouldn't get done. And the reason I believe CSS would be a Peacock is because Peacocks are bright and colorful, which is exactly what CSS delivers to web development and design.
