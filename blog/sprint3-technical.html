What's The Dom?

The DOM stands for the Document Object Model

D = Document = A file (which in our case is an HTML file, just like a .doc file, mp4, mp3, jpeg, etc.

O = Object = the objects within the file = the tags within our HTML file = htmls tags, header tags, body tags, button tag, and more...

M = Model = How you layout all of these structures. E.g. html tags, head tags, body tags, title tags, etc. all laid out properly and all of their different levels.

In short, the DOM is the model of all of the objects within the docoment(s) you're looking to display. For example, if you have an HTML document, then the DOM for that particular file would be the html tags over the head and body tags which wrap the child tags within themselves.

The Difference Between Accessing Data From Arrays and Object

Arrays is a list of certain variables, i.e. names, or ages, or heights, or weights, etc. While objects give you multiple variables and data values, i.e. name, and age, and height, and weight, etc.

Functions and Why They're Useful
